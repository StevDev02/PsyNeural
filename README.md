## API-PSYNEURAL

Simple GraphQL Blog API using Nodejs and Mongodb

### Requerimentos

- Mongodb
- Nodejs

### Environment variables

```
  DEV_URI_DB
  PROD_URI_DB
  PORT
  JWT_SECRET
  SECRETDECRIPT
  ADMIN_USER
  LOCAL_ADMIN_USER
  JWT_EXPIRES_IN
  STATE
```

### Installation

```
npm i
npm start
```
